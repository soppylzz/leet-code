# LeetCode 刷题训练<sup style="font-size: 0.5em">[GPT Text]</sup>
个人专用 LeetCode 算法题训练仓库，用于记录刷题过程、整理解题思路与代码实现。

## 仓库说明
1. **自用训练仓库**：本仓库为个人 LeetCode 刷题专属项目，持续更新算法题解题代码与题解。
2. **题目难度标识规则**
    - 简单题：`[题目编号][]`（无后缀标识）
    - 中等题：`[题目编号][m]`
    - 困难题：`[题目编号][h]`
    - 示例：`[001][]` 表示第1题简单难度，`[002][m]` 表示第2题中等难度
3. **特殊算法标识规则**
   解题使用特殊算法、优化思路或进阶解法时，追加 `[a]` 标识
    - 示例：`[146][m][a]` 表示第146题中等难度，且使用特殊算法实现
